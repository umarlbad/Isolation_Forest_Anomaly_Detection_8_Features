# Isolation_Forest_Anomaly_Detection_8_Features
Build an Anomaly Detection based on Isolation Forest Algorithm with 8 Features for developing a Fraud Detection System's Engine.

## Overview
This programme is used to compare the results of anomaly predictions previously carried out by Fast-MCD (Minimum Covariance Determinant) using the same threshold. By incorporating this ML algorithm, it is hoped that the Fraud Detection System (FDS) will achieve maximum accuracy in identifying fraud. The FDS comprises several ML algorithms, one of which is the Isolation Forest.
